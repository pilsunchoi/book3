2 회귀분석 대상 데이터 분석#

변수 설명#

변수명

내     용

score

5개 과목(국어, 영어, 수학, 사회, 과학) 성적 평균값. 개별 과목의 성적은 5점 척도(1. 매우 못했다, 2. 못한 편이다, 3. 보통이다, 4. 잘한 편이다, 5. 매우 잘 했다)로 평가됨.

wave

조사차수: 1. 2010년, 2. 2011년, 3. 2012년, 4. 2013년, 5. 2014년, 6. 2015년, 7. 2016년

mc

다문화가정 더미변수: 국제결혼 가정, 외국인 노동자 가정, 새터민(탈북자) 가정, 조선족 가정 등. 0. 아니오, 1. 예

male

남학생 더미변수: 0. 여학생, 1. —남학생

income

가구 소득수준: 연소득, 세후, 단위: 1,000만 원

fa_edu / mo_edu

아버지 및 어머니 학력: 1. 중졸이하, 2. 고졸, 3. 전문대졸, 4. 대졸, 5. 대학원졸, 6. 해당사항 없음 (함께 살고 있지 않은 경우 “해당사항 없음”으로 응답)

par_satis

보호자 삶의 만족도: 1. 전혀 만족하지 못한다, 2. 만족하지 못하는 편이다, 3. 보통이다, 4. 매우 만족한다

health

본인 건강상태: 1. 매우 건강하지 못하다, 2. 건강하지 못한 편이다, 3. 건강한 편이다, 4. 매우 건강하다

aft_sch

학원・과외시간: 하루 평균 학원 및 과외 시간(단위: 시간). 등교일과 비등교일을 2대 1의 비율로 가중평균함

aft_sch1

방과후교육 경험 더미변수: 0. 아니오, 1. 예

presch

초등 전 사교육 경험 더미변수: 0. 아니오, 1. 예

care
(부모의 관심)

- 다른 일(직장이나 바깥일)보다 나를 더 중요하게 생각하신다
- 내가 학교에서 어떻게 생활하는지 관심을 갖고 물어보신다
- 내 몸이나, 옷, 이불 등이 깨끗하도록 항상 신경 쓰신다
- 내가 많이 아프면 적절한 치료를 받게 하신다

abuse
(부모의 학대)

- 내가 무언가 잘못했을 때 정도 이상으로 심하게 혼내신다
- 내가 잘못하면 무조건 때리려고 하신다
- 내 몸에 멍이 들거나 상처가 남을 정도로 나를 심하게 대하신 적이 많다
- 나에게 심한 말이나 욕을 하신 적이 많다

sch_study
(학교 학습활동)

- 학교 수업 시간이 재미있다
- 학교 숙제를 빠뜨리지 않고 한다
- 수업 시간에 배운 내용을 잘 알고 있다

friend
(교우 관계)

- 우리 반 아이들과 잘 어울린다
- 친구와 다투었을 때 먼저 사과한다
- 친구가 교과서나 준비물을 안 가져왔을 때 함께 보거나 빌려준다
- 친구가 하는 일을 방해한다
- 놀이나 모둠활동을 할 때 친구들이 내 말을 잘 따라준다

teacher
(교사 관계)

- 선생님을 만나면 반갑게 인사한다
- 선생님과 이야기하는 것이 편하다
- 학교 밖에서 선생님을 만나면 반갑다
- 우리 선생님께서는 나에게 친절하시다

주:

care, abuse, sch_study, friend, teacher 등은 표의 해당 문항들에 대한 응답을 평균한 것으로서, 모든 문항들은 동일한 4점 척도(1=전혀 그렇지 않다, 2=그렇지 않은 편이다, 3=그런 편이다, 4=매우 그렇다)로 조사됐다.

4 두 그룹의 평균 비교#

본 연구에서 우리의 관심은 이와 같은 학업성적 차이가 대상 학생을 둘러싸고 있는 여러 배경 및 발달 변인을 동일하게 통제한 상태에서도 여전히 존재하는지 여부이다. 그것을 패널 회귀분석 모형으로 추정하지만, 간단히 두 그룹의 평균 성적 차이에 대한 통계적 유의성을 \(t\)-검정으로 추정해볼 수 있다.

우리의 데이터가 패널 자료라는 점을 감안하지 않고, 각 패널에 대해 모든 시점 별로 \(t\)-검정을 실시한 결과, 1% 유의수준에서 차이가 나는 경우는 총 17개 시점에 대한 검정에서 단 하나도 발견되지 않았다. 유의수준을 5%로 할 경우, 초1・초4 패널 각 6개 시점 중 단 1개씩에서만 평균 성적의 차이가 유의했고, 중1 패널의 경우 5개 시점 중 2개 시점에서 평균값이 유의하게 달랐다.

초1 패널 평균차이에 대한 \(t\) 검정#

import scipy.stats as stats

# 결과를 저장할 리스트
t_test_results = []

# 각 WAVE에서 다문화학생과 비다문화학생의 평균값 차이 검정
for wave in df_e1['WAVE'].unique():
    # 비다문화학생 score
    non_mc_scores = df_e1[(df_e1['mc'] == 0) & (df_e1['WAVE'] == wave)]['score'].dropna()
    # 다문화학생 score
    mc_scores = df_e1[(df_e1['mc'] == 1) & (df_e1['WAVE'] == wave)]['score'].dropna()
    
    # t-검정 실행 (동분산 가정 여부는 데이터에 따라 결정)
    t_stat, p_value = stats.ttest_ind(non_mc_scores, mc_scores, equal_var=False)
    
    # 결과 저장
    t_test_results.append({
        'WAVE': wave,
        't-statistic': t_stat,
        'p-value': p_value,
        'non_mc_mean': non_mc_scores.mean(),
        'mc_mean': mc_scores.mean(),
        'non_mc_count': len(non_mc_scores),
        'mc_count': len(mc_scores)
    })

# 결과를 데이터프레임으로 변환하여 확인
t_test_df = pd.DataFrame(t_test_results)

# 새로운 인덱스 값 설정
new_index = ['초2', '초3', '초4', '초5', '초6', '중1']
# 인덱스를 새로운 값으로 변경
t_test_df.index = new_index

print(t_test_df)
